cas (LDAP Integation)
=====================

The ldapcas module extends the functionality of cas
(http://drupal.org/project/cas) to allow fetching of
LDAP user data when a user logs in via CAS.

Required Modules:
-----------------
CAS - http://drupal.org/project/cas
